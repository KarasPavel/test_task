Для пользователей unix систем:
1. Открыть терминал
2. выполнить в терминале команду > mysql -u Имя пользователя -p Название Базы данных < путь к скрипту review.sql
3. Ввести пароль
Для пользователей ide phpstorm:
1. Подключится к базе данных
2. ПКМ по названию вашей БД > Run SQL Script... > путь к файлу review.sql > execute
